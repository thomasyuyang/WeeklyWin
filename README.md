# SMH + NVDA Trade Timing App

This Streamlit app is designed for cloud deployment through GitHub + Streamlit Community Cloud.

## Files

- `smh_nvda_trade_timing_app.py` — main Streamlit app
- `requirements.txt` — Python dependencies
- `.streamlit/config.toml` — Streamlit theme and cloud/mobile-friendly settings

## Deploy to Streamlit Community Cloud

1. Create a GitHub repository, for example:
   `smh-nvda-trade-timing-app`

2. Upload these files to the root of the repository:

   - `smh_nvda_trade_timing_app.py`
   - `requirements.txt`
   - `.streamlit/config.toml`

3. Go to Streamlit Community Cloud:
   `https://share.streamlit.io`

4. Click **Create app** or **Deploy an app**.

5. Select your GitHub repository.

6. Set the main file path to:

   `smh_nvda_trade_timing_app.py`

7. Click **Deploy**.

## Use on Android phone and computer

After deployment, Streamlit gives you a link like:

`https://your-app-name.streamlit.app`

Open that link on:

- Android phone Chrome
- Computer browser
- Tablet browser

On Android, you can add it to your home screen:

Chrome → three dots → Add to Home screen.

## Notes

This app is educational decision support only. It does not place trades and does not guarantee profit.
